# financial-performance-dashboard

## Business Problem
This project analyzes revenue, expenses, and profit trends for a fictional company.

## Tools Used
- Excel
- SQL
- Power BI
- Python

## Key Insights
- Sales and profit performance vary across products, countries, and customer segments.
- Discount levels appear to affect profit margin, making discount strategy important for financial performance.
- Monthly sales trends help identify stronger and weaker financial periods.
- The dashboard is designed to help leadership quickly review revenue, profit, and business performance.
